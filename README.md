# Brandon Chat - Aplicación de Chat con Multimedia

Aplicación de chat en tiempo real con soporte para imágenes, audio y video, optimizada para dispositivos móviles.

## Características

- 💬 **Chat en tiempo real** con Firebase Realtime Database
- 📷 **Cámara integrada** para tomar fotos sin recorte
- 🎤 **Grabación de audio** con visualizador en tiempo real
- 📁 **Subida de imágenes** desde galería
- 🔄 **Respuestas a mensajes** con preview
- ✏️ **Edición de mensajes** (pulsación larga)
- 🌐 **Optimizado para móviles** con soporte para área segura (notch)
- ☁️ **Subida a Cloudinary** con barra de progreso
